## PennyWise

Simple API for facilitating saving for users.

## Installation
To install PennyWise, follow these simple steps:

1. clone the repo:
    ```
    git clone [repo url]
    ```
2. Install dependencies
   ```
   yarn install
    ```
3. Start the development server:
    ```
    yarn start:dev
    ```

You should have PennyWise installed...
